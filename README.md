# SCT_SD_2

## Task 2 - Guess The Number Game

This is a GUI-based number guessing game built using Python Tkinter.

### Features:
- Random number generation (1–100)
- User input with validation
- Feedback (Too High / Too Low / Correct)
- Restart option
- Clean UI with background image

### Technologies Used:
- Python
- Tkinter
- Pillow (for background image)
